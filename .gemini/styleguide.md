# Gemini Code Assist Instructions — NFCWorkFlow (MSc Project 25-26)

## Pull Request Guidelines (Summary & Review)

> **Important:** The following rules must be strictly followed for ALL interactions.

1. **Language (Mandatory):** - **PR Summary:** Always generate the pull request summary (the high-level overview) in **Simplified Chinese (简体中文)**.
   - **Code Review:** For every pull request review, always write the review comments in **Simplified Chinese (简体中文)**.
   - Do not use English for any part of the output unless the user explicitly requests it in a comment.

2. **Summary Requirements:**
   - Summarize in Simplified Chinese (简体中文).

3. **Change Explanations (Review):** - If a suggested change is given, explain in detail why this action was taken in Chinese, including:
     - What problem the change solves
     - Why this approach was chosen
     - Impact on quality, performance, or security
